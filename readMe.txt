Цей скрипт допоможе автоматично встановити та налаштувати proxychain,  
	а також встановити та запустити ТОR сервіс.
	Також можна в будьякий час додати свій проксі в proxychain.

[ ЗВЕРНІТЬ УВАГУ - ДАННИЙ СКРИПТ ПРАЦЮЄ ТІЛЬКИ НА LINUX І ПОТДІБНИХ ДО НЬОГО СИСТЕМАХ ]

>>> ЗАПУСКАТИ ДАННИЙ СКРИПТ ТІЛЬКИ З ПРАВАМИ АДМІНІСТРАТОРА (sudo-)
>>> ПЕРЕД ЗАПУСКОМ ПЕРЕВІРТЕ ЧИ У ВАС ВСТАНОВЛЕНИЙ PYTHON3, ЯКЩО НІ, ТО > sudo apt install python3 

> Запуск: sudo python3 meanon.py

[@] Перед вами меню з функціями.
1. Installer > За допомогою нього можна автоматично встановити на вибір потрібні скрипти та програми:
	1. proxychains
	2. TOR service
	3. MHDDoS (Директорія з скриптом - /MHDDoS)
		> Інструкція MHDDoS - https://github.com/MHProDev/MHDDoS

2. Edit proxychains > Тут можна відредагувати список проксі в proxychains:
	1. Add your own proxies. - Додати власні проксі. Проксі вписуєте через кому [PROXY TYPE] [IP] [PORT],...,[PROXY TYPE] [IP] [PORT]
	2. Add new proxies from [FILE].txt. - Mожна додати список проксі в proxychains з файлу(назву файлу та тип проксі в ньому потрібно ввести)
		[!] Проксі в списку мають бути одного типу, який ви ввели перед тим. 
		[!] Фйл має знаходитись в директорії скрипта
		> Список в файлі має мати такий вигляд:
			/------------------------------------------------------------\
			| 			[FILE].txt		             |
			|------------------------------------------------------------|
			|[IP]:[PORT]						     |
			|[IP]:[PORT]						     |
			|.....							     |
			|[IP]:[PORT]						     |
			|							     |
			\------------------------------------------------------------/
		[!] При неправильному записі proxychains буде працювати некоректно.
	3. Delete all proxies from proxychains. - Oчистити список проксі в proxychains

3. TOR management > Керування ТОР сервісом (1 - старт,2 - стоп).

> Вибирайте потрібну фунцію, по ходу роботи скрипт повідомляє вас як і що треба вводити.

[!] Що робити коли викинуло або показало помилку:
	- Перевірте чи скрипт запущено з правами адміністратора
	- Перевірте чи немає запущеного процесу apt-get/apt, якщо такий є, закрийте його, або ще краще перезапустіть машину
	- Перевірте підключення до інтернету
	

[>] По всіх питаннях та проблемах пишіть мені в телеграм(вказаний зверху).
